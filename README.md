Usage

1. Install dependencies:

```bash
pip install pandas yfinance
```

2. Run the script (default reads the sample file on your Desktop):

```bash
python c:\Users\Abhishekh\OneDrive\Desktop\portfolio_report.py
```

Or provide a custom trade file path:

```bash
python c:\Users\Abhishekh\OneDrive\Desktop\portfolio_report.py path\to\trades.csv
```

Output

- Prints a simple table to the console.
- Writes `portfolio_report.csv` in the current working directory.
